OPENQUANT · DESIGN SYSTEM · v1.0

为量化盯盘而设计的视觉系统

这是 OpenQuant 的设计语言——纸质触感的中性色、克制的版面节奏、以青色/品红替代传统红绿的涨跌色。目标是：长时间盯盘不疲劳、数字可读、科技感克制且友好、信息熵降到最低。

08原则 · Principles

熵减
每多一个元素都要为信息增益负责。去掉装饰性渐变、多余 emoji、不必要的 badge。
克制
颜色只在涨跌、警告、当前选择时出现。其余一律中性。让数字说话。
等宽对齐
所有数字使用 tabular-nums + SF Mono，跳动时不撑行、不错位。
动画即信号
只有"有意义的变化"才动——tick 闪烁、大涨大跌数字放大、通知滑入。
不预设情绪
避开视网膜刺激色（饱和红绿）。青/品红让用户自己做判断，不煽动交易。
纸与屏
纸色底 + 衬线大标题，给信息密集的工作面留一点呼吸。像读一份早报。
