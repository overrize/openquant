# 贵金属·原油 数据来源与常规单位

## 当前实现（东方财富）

- **数据来源**：**东方财富** push2 接口（`push2.eastmoney.com/api/qt/stock/get`），国内期货/次主连行情，**无需 API Key**。
- **品种与单位**（与国内期货一致）：

| secid    | 名称         | 单位     |
|----------|--------------|----------|
| 113.aus  | 沪金次主连   | 元/克    |
| 113.ags  | 沪银次主连   | 元/千克  |
| 142.scs  | 原油次主连   | 元/桶    |

- **说明**：113 = 上期所，142 = 上期能源；价格与涨跌幅直接为接口返回的常规单位，无折算。

---

## 其他可选 API（常规单位）

若希望直接用 **美元/盎司、美元/克、美元/桶** 等接口，可考虑：

| 来源 | 说明 | 单位 |
|------|------|------|
| [MetalpriceAPI](https://metalpriceapi.com/) | 贵金属现货，免费额度有限 | 美元/盎司，可自行换算为 /克 |
| [ExchangeRate-API](https://www.exchangerate-api.com/) | 部分计划含黄金等 | 美元/盎司 |
| [Alpha Vantage](https://www.alphavantage.co/) | 有 Commodity 等接口 | 需查文档 |
| [Twelve Data](https://twelvedata.com/) | 股票/外汇/商品 | 多种 |
| 交易所/数据商 | 如 COMEX 黄金、WTI 期货 | 美元/盎司、美元/桶 |

接入时建议**统一用常规单位**（如美元/克、美元/桶）写入与展示，便于和当前「按克、按桶」的展示方式一致。

## App 安装（PWA）

项目已加 `manifest.json` 和主题色等，部署到 **HTTPS** 后：

- 浏览器可「添加到主屏幕」当 App 使用。
- 图标需自备：在 `public/` 下放置 `icon-192.png`、`icon-512.png` 即可；未放置时部分浏览器仍可安装，仅无自定义图标。
